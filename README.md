# Actividad-Grupal-R - Proyecto Transversal en R (I): Análisis de bioestadística aplicada a la expresión génica en Los Simpson 

Se utiliza un dataset que contiene información de la expresión de 37 genes en 59 pacientes de la población Los Simpson, cada uno con distintas características sociodemográficas y de expresión génica. Para ello realizarás las siguientes tareas:

▸	Abrir la base de datos.
▸	Análisis de normalidad.
▸	Análisis de componentes principales (PCA).
▸	Visualización.
▸	Agrupación de individuos (clustering).
▸	Análisis descriptivo estratificado por terciles.
▸	Modelo predictivo (regresión logística).
